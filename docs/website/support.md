# Support

Need help with Immich Lounge?

If you need Roku beta access before the public store release, get in touch and ask for a beta code.

## Before Opening an Issue

Please check:

- [Getting Started](./getting-started.md)
- [Installation](./installation.md)
- [Configuration](./configuration.md)
- [Troubleshooting](./troubleshooting.md)

## GitHub Issues

Use GitHub Issues for:

- reproducible bugs
- feature requests
- documentation improvements

When reporting a problem, include your Roku model, Roku OS version, immich version, and whether the issue is in the companion app, channel, or screensaver.
